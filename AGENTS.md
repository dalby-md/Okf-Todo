## Frontend

- Do not add npm, Vite, Vue, React, or a bundler unless explicitly requested.
- Use local `Okf-Todo/wwwroot/js/jquery.min.js`.
- Put app code in `Okf-Todo/wwwroot/js/app.js`.
- Put styles in `Okf-Todo/wwwroot/css/app.css`.
